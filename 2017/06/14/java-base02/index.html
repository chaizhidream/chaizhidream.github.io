关键代码为：

华氏度变成摄氏度：

double f; // 华氏温度
System.out.println((f – 32.0) * 5 / 9);

摄氏度转成华氏度

double c ; // 摄氏温度
System.out.println(c * 9.0 / 5 + 32);
